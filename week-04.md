# 第4-7周：Docker 容器技术

---

## 📋 本周基本信息
- **学时**：18学时（第4-7周）
- **实验类型**：综合性实验
- **前置知识**：Linux 基础

---

## 🎯 本周学习目标

通过 Docker 专业实验，使学生：

### 知识目标
- 理解容器化技术的发展趋势和基本原理
- 掌握 Docker 的架构和核心概念
- 掌握 Docker 镜像、容器、仓库的使用方法
- 熟悉 Docker Hub 的使用
- 理解 Docker Compose 的基本用法

### 能力目标
- 能够独立安装和配置 Docker 环境
- 能够使用 Docker 运行 Linux 环境、MySQL、Python 等
- 能够编写简单的 Dockerfile
- 能够进行容器的端口映射和数据卷挂载

### 素质目标
- 培养研究性和探索性学习方式
- 提升自主学习能力
- 培养容器化思维

## 🛠️ 实战任务

### 任务一：Docker 环境搭建
- 安装 Docker Desktop
- 运行 hello-world 镜像
- 配置 Docker 镜像源

### 任务二：容器基本操作
- 运行 Ubuntu 容器
- 容器的启动、停止、删除
- 进入容器进行操作

### 任务三：镜像管理
- 搜索和拉取镜像
- 查看和删除镜像
- 保存和加载镜像

### 任务四：数据持久化
- 数据卷的使用
- 绑定挂载
- 端口映射

### 任务五：服务部署
- 运行 MySQL 容器
- 运行 Python 环境
- 多容器协作

## ⚠️ 常见问题
- Docker 启动失败？检查虚拟化是否开启
- 拉取镜像慢？配置国内镜像源
- 端口映射不生效？检查端口是否被占用
