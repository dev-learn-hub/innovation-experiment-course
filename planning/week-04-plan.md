## ⚠️ 常见问题与解决方案

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| Docker Desktop 无法启动 | 虚拟化未开启 | 在 BIOS 中开启虚拟化，Windows 开启 WSL2/Hyper-V |
| 拉取镜像很慢 | 未配置镜像源 | 配置国内镜像源（中科大、网易等） |
| 容器退出数据丢失 | 未使用数据卷 | 使用 -v 参数挂载数据卷或绑定主机目录 |
| 端口映射不生效 | 端口被占用或容器内服务未监听 | 检查主机端口占用，确保容器内服务监听 0.0.0.0 |
| 无法进入容器 | 容器未运行或 shell 不存在 | 确保容器在运行，使用 /bin/bash 或 /bin/sh |
