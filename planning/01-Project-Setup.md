# 项目方案：创新实验课课程开发

## 整体目标

为《创新实验课》开发完整的教学内容，让学生可以：
- 清晰了解课程目标和每周安排
- 通过演示案例理解知识点
- 通过习题案例自己动手练习巩固
- 跟着讲义步骤一步步操作

## 目录结构设计（模仿 flutter-course）

```
innovation-experiment-course/
├── index.md                # 课程大纲（总览）
├── week-XX.md              # 每周聚合页
├── week-XX-lecture.md      # 每周讲义
├── week-XX-exercise.md     # 每周练习
├── week-XX-page.md         # 每周详情页
├── demos/                  # 所有教学案例
│   ├── week-03/
│   │   ├── demo/           # 老师演示案例
│   │   └── exercise/       # 学生练习案例
│   └── ...
├── planning/               # 项目方案设计
│   ├── 01-Project-Setup.md（本文档）
│   ├── week-XX-plan.md     # 每周规划
│   └── week-XX-knowledge-map.md  # 每周知识图谱
└── README.md
```
