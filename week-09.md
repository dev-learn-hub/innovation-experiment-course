## 🎯 本周学习目标

通过 Flutter 专业实验，使学生：

### 知识目标
- 掌握 Flutter 框架的基本原理和跨平台特性
- 掌握 Dart 编程语言的语法和特性
- 掌握 Widget 体系、状态管理、路由导航等核心知识
- 熟悉 Flutter 开发环境的搭建和使用
- 了解跨平台应用开发的基本流程

### 能力目标
- 能够独立搭建 Flutter 开发环境
- 能够使用官方文档和示例代码进行学习
- 能够开发简单的跨平台 UI 应用
- 能够使用 GitHub 进行版本控制和协作
- 能够开发和改进一个完整的应用项目

### 素质目标
- 培养跨平台思维和多端适配意识
- 培养用户体验设计意识
- 培养开源贡献和社区参与意识

## 🛠️ 实战任务

### 任务一：Flutter 环境搭建
- 安装 Flutter SDK
- 配置 Android Studio / VS Code
- 运行 Flutter Doctor 检查环境
- 创建第一个 Flutter 应用

### 任务二：Dart 语言基础
- 变量、数据类型、函数
- 面向对象编程
- 异步编程（Future、async/await）

### 任务三：Widget 开发
- StatelessWidget 和 StatefulWidget
- 常用布局组件（Row、Column、Stack、ListView）
- 常用 UI 组件（Button、TextField、Image、Card）

### 任务四：状态管理与导航
- setState 基础状态管理
- 页面路由与导航
- 简单的状态管理方案（Provider）

### 任务五：网络与数据
- HTTP 请求（http 包）
- JSON 序列化与反序列化
- 本地存储（shared_preferences）

### 任务六：完整项目开发
- 需求分析与原型设计
- 项目架构与代码组织
- 功能实现与测试
- GitHub 仓库管理与协作

## ⚠️ 常见问题
- Flutter 环境配置失败？仔细阅读 Flutter Doctor 的提示
- 组件不显示？检查 Widget 树结构和约束
- 状态不更新？确保使用了 setState 或状态管理方案
