# 第8周：Web 开发与 Jekyll

---

## 📋 本周基本信息
- **学时**：6学时
- **实验类型**：设计性实验
- **前置知识**：HTML、Markdown 基础

---

## 🎯 本周学习目标

通过 Web 专业实验，使学生：

### 知识目标
- 掌握 Jekyll 静态网站生成器的基本原理
- 掌握 Markdown 内容编写方法
- 理解 Jekyll 的目录结构和模板系统
- 掌握 GitHub Pages 的部署方法

### 能力目标
- 能够使用 Jekyll 创建和管理个人博客
- 能够编写 Markdown 格式的内容
- 能够使用 GitHub Pages 或 Nginx 部署网站

### 素质目标
- 培养技术写作和内容整理能力
- 培养知识分享意识

---

## 📚 本周核心内容

| 模块 | 内容 | 重点 |
|------|------|------|
| 1 | Jekyll 基础 | 安装、基本概念、目录结构 |
| 2 | Markdown 写作 | Markdown 语法、内容组织 |
| 3 | 模板与主题 | 布局、包含文件、变量 |
| 4 | 部署上线 | GitHub Pages、Nginx 部署 |

---

## 🛠️ 实战任务

### 任务一：Jekyll 环境搭建
- 安装 Ruby 和 Jekyll
- 创建新的 Jekyll 站点
- 本地预览网站

### 任务二：Markdown 内容创作
- 编写 Markdown 格式的文章
- 添加图片、链接、代码块
- 组织内容分类和标签

### 任务三：模板定制
- 修改网站布局
- 添加导航栏和页脚
- 自定义样式

### 任务四：网站部署
- 使用 GitHub Pages 部署
- 或使用 Nginx 部署
- 绑定域名（可选）

## ⚠️ 常见问题

- 网站生成错误？检查 Front Matter 格式
- GitHub Pages 不更新？检查仓库设置
