# Docker 练习模板

本目录包含 Docker 课程的练习模板，学生需要在此基础上完成练习。

## 练习列表

| 练习 | 名称 | 说明 |
|------|------|------|
| 01 | Hello World | 验证 Docker 安装 |
| 02 | Ubuntu | 容器基础操作 |
| 03 | Nginx | 端口映射 |
| 04 | MySQL | 数据卷持久化 |
| 05 | Python | 脚本运行 |

## 使用说明

1. 每个练习目录包含 README.md 和必要的模板文件
2. 学生按照 README.md 中的要求完成任务
3. 完成后截图或记录输出结果
4. 提交到指定位置

## 注意事项

- 不要修改 demo/ 目录，那是完整示例
- 在 exercise/ 目录完成你的练习
- 有问题先看 demo/ 目录的示例
