# Flutter 演示示例

本目录包含 Flutter 课程的完整演示代码。

## 示例列表

| 示例 | 名称 | 说明 |
|------|------|------|
| 01 | Hello World | 基础 Flutter 应用 |
| 02 | 计数器 | StatefulWidget 和状态管理 |

## 前置要求

- Flutter SDK 已安装
- 可用的开发设备（模拟器、真机或 Web）

## 快速开始

进入任意示例目录，执行：

```bash
flutter pub get
flutter run
```

## 常用命令

```bash
# 查看可用设备
flutter devices

# 运行到特定设备
flutter run -d <device_id>

# Web 浏览器
flutter run -d chrome

# Windows 桌面
flutter run -d windows

# 查看日志
flutter logs

# 清理构建
flutter clean
```

## 学习路径

1. **01-hello-world** - 了解基本结构
2. **02-counter** - 学习状态管理
3. 修改代码，体验 Hot Reload（按 r）
