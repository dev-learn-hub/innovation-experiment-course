# 练习 1：创建我的博客

## 任务

1. 修改 `_config.yml`，把 `title` 改成你的博客名字
2. 修改 `index.md`，写一段关于你自己的介绍
3. 在 `_posts` 目录添加一篇新文章（文件名格式：YYYY-MM-DD-title.md）
4. 把这个目录推送到 GitHub，用 GitHub Pages 上线你的博客！

## 要求

- [ ] 博客标题已修改
- [ ] 首页有自我介绍
- [ ] 至少有 2 篇文章
- [ ] 成功发布到 GitHub Pages

## 提示

### 文章文件名格式

```
YYYY-MM-DD-文章标题.md
```

例如：`2026-03-18-my-third-post.md`

### 发布到 GitHub Pages

1. 在 GitHub 创建一个仓库
2. 把这个目录的内容推送到仓库
3. 进入仓库 Settings → Pages
4. 选择 main 分支，点击 Save
5. 等待几分钟，访问 `https://你的用户名.github.io/仓库名/`
