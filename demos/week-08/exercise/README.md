# Jekyll 练习模板

本目录包含 Jekyll 课程的练习模板。

## 练习列表

| 练习 | 名称 | 说明 |
|------|------|------|
| 01 | 我的博客 | 创建第一个 Jekyll 博客 |

## 使用说明

1. 进入练习目录（如 `01-my-blog`）
2. 按照 README.md 中的要求完成任务
3. 完成后推送到 GitHub，用 GitHub Pages 上线
