# 云南大学《创新实验》课程

---

## 📋 课程简介

《创新实验》是云南大学智能科学与技术专业的综合实验课程，共 2 学分，54 学时。

本课程通过 5 个专业实验，培养学生的创新意识和实践能力、科研基本能力和知识更新能力，使学生掌握开源软件/硬件的基本原理和应用方法。

---

## 🎯 课程目标

- 培养研究性和探索性学习方式
- 提升自主学习能力
- 培养创新意识和团队协作能力
- 接触专业科研开发领域和工业实践

## 💡 课程特色

1. **研究性学习**：在开源软件/硬件平台上以研究和探索的方式学习
2. **产学研结合**：营造教学与科研紧密结合的氛围
3. **多技术栈**：涵盖机器人对战、容器技术、Web开发、移动应用开发
4. **团队协作**：通过小组项目培养团队协作和沟通能力
