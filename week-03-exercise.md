## 基础练习（必做）

### 练习一：HelloWorld 机器人
**目标**：编写第一个可运行的机器人

**要求**：
1. 创建一个名为 `HelloWorld` 的机器人
2. 实现以下功能：
   - 向前移动 100 像素
   - 右转 90 度
   - 循环重复上述动作
3. 扫描到敌人时开火（火力 1.0）

**提示**：
```java
public class HelloWorld extends Robot {
    public void run() {
        while (true) {
            ahead(100);
            turnRight(90);
        }
    }
    
    public void onScannedRobot(ScannedRobotEvent e) {
        fire(1);
    }
}
```

**验收标准**：
- [ ] 机器人能正常编译
- [ ] 机器人能在对战中运行
- [ ] 机器人能按正方形轨迹移动
- [ ] 机器人能扫描并射击敌人

### 练习二：改进移动策略
**目标**：让机器人移动更难以预测

**要求**：
1. 创建一个名为 `RandomMover` 的机器人
2. 实现以下功能：
   - 每次移动随机距离（50-200 像素）
   - 每次转向随机角度（-180 到 180 度）
   - 移动时保持炮管旋转扫描
3. 扫描到敌人时根据距离调整火力

**提示**：
```java
public void run() {
    while (true) {
        ahead(50 + Math.random() * 150);
        turnRight(-180 + Math.random() * 360);
        turnGunRight(360);
    }
}
```

**验收标准**：
- [ ] 机器人移动轨迹不规则
- [ ] 炮管保持持续扫描
- [ ] 能根据距离调整火力大小

## 进阶练习（选做）

### 练习三：目标锁定机器人
**目标**：让炮管始终指向敌人

**要求**：
1. 创建一个名为 `TargetLock` 的机器人
2. 实现以下功能：
   - 扫描到敌人后，计算敌人相对于炮管的角度
   - 旋转炮管指向敌人
   - 当炮管对准敌人时开火
3. 保持机器人移动，避免被击中

**提示**：
```java
public void onScannedRobot(ScannedRobotEvent e) {
    double absoluteBearing = getHeading() + e.getBearing();
    double bearingFromGun = normalRelativeAngle(absoluteBearing - getGunHeading());
    
    if (Math.abs(bearingFromGun) <= 3) {
        fire(2);
    }
    turnGunRight(bearingFromGun);
}
```

**验收标准**：
- [ ] 炮管能持续追踪敌人
- [ ] 只在瞄准后开火
- [ ] 机器人保持移动

### 练习四：墙壁躲避机器人
**目标**：让机器人避免撞墙

**要求**：
1. 创建一个名为 `WallAvoider` 的机器人
2. 实现以下功能：
   - 检测与墙壁的距离
   - 当靠近墙壁时转向
   - 保持在战场中央区域活动
3. 同时保持扫描和射击

**提示**：
- 使用 `getX()` 和 `getY()` 获取位置
- 战场大小：800x600（默认）
- 保持距离墙壁 50 像素以上

**验收标准**：
- [ ] 机器人不会撞到墙壁
- [ ] 能在战场中自由移动
- [ ] 保持扫描和射击功能
